# WAVPlayer_s1131547

##  核心功能介紹
* **1. 瀏覽與選擇檔案**：
  點擊「瀏覽」按鈕後，會跳出選擇檔案的視窗。選好音樂後，檔案的儲存路徑會自動顯示在畫面上的文字框裡，讓使用者清楚知道現在準備播放哪一首歌。
* **2. 單次播放**：
  點擊「播放」按鈕，程式會讀取文字框裡的檔案路徑，並將該首音樂完整地播放一次。
* **3. 循環播放**：
  點擊「循環播放」按鈕，音樂會不斷重複播放，不會中斷，可以用來播放背景音樂。
* **4. 停止播放**：
  在音樂播放的過程中，無論是單次播放還是循環播放，只要隨時點擊「停止」按鈕，音樂就會立刻中斷。
  
##  防呆與提示機制

**1. 限定檔案格式 (避免選錯檔案)：**
為了確保音樂能順利播放，在點選「瀏覽」挑選檔案時，系統已經自動設定好「只能看見並選擇 `.wav` 格式」的音訊檔。這可以防止使用者不小心點到 mp3 或圖片檔，導致程式出錯無法執行。

**2. 關閉程式雙重確認 (防止誤觸)：**
為了防止使用者不小心按到右上角的「X」或是畫面上的「結束程式」按鈕，導致聽到一半的音樂突然關閉，系統特別加入了防呆提示。在關閉前會跳出「確定要關閉應用程式嗎？」的視窗，如果是不小心點到的，只要選「否」就能取消關閉，繼續聽音樂。
